■本アーカイブについて
  　本アーカイブは「プロになるための Web 技術入門」内で紹介したサンプル
  ソースコードを配布するものです。
  
■本アーカイブの構成
  　以下のようなフォルダ構成となっています。
  
  ├ mock              Lesson4 ピザペントミノ画面モック
  ├ pentomino_1       Lesson4 ピザペントミノ(PHP) 認証機能のみ
  │   (http://www.littleforest.jp/webtext/pentomino_1/login.html から確認可能)
  │
  ├ pentomino_2       Lesson4 ピザペントミノ(PHP) Cookie使用版
  │   (http://www.littleforest.jp/webtext/pentomino_2/login.html から確認可能)
  │
  ├ pentomino_3       Lesson4 ピザペントミノ(PHP) 完成版
  │   (http://www.littleforest.jp/webtext/pentomino_3/login.html から確認可能)
  │
  ├ PizzaPentomino    Lesson6 ピザペントミノ(Java) Struts版
  │   (http://www.littleforest.jp/webtext/PizzaPentomino/ から確認可能)
  │
  └ PizzaPentominoSrv Lesson6 ピザペントミノ(Java) サーブレット版
  
  ※ Java 版については Eclipse へインポート可能な形式となっています。
